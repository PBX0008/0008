# NCLEX Dose Calculations Study Guide

A mobile-friendly static website for learning NCLEX RN and NCLEX PN medication and dose calculations.

## Included files

- `index.html` — main page
- `assets/css/styles.css` — responsive styling
- `assets/js/script.js` — calculators, practice answer toggles, search, and mobile menu
- `LICENSE` — MIT license

## What is included

- Step-by-step teaching
- Worked examples
- Built-in calculators
- Practice questions
- Mobile-friendly navigation
- GitHub Pages ready structure

## Upload to GitHub

1. Create a new repository on GitHub.
2. Upload all files and folders from this project.
3. Make sure `index.html` stays in the root of the repository.
4. Commit the files.

## Publish with GitHub Pages

1. Open your repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save.

GitHub will publish the site and give you a live URL.

## Local preview

You can open `index.html` directly in a browser, or use a local server.

Example with Python:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Editing tips

- Change text content in `index.html`
- Change colors, spacing, and layout in `assets/css/styles.css`
- Change calculator behavior in `assets/js/script.js`
